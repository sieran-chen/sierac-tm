# 罐装机多角度图片

将 `3d/罐装机图片/` 下 14 张 PNG 按视角重命名后放入此目录。

## 文件名与视角对应（design §2.2）

| 文件名 | 视角说明 |
|--------|----------|
| front.png | 正前方 |
| front-right.png | 右前方 45° |
| right.png | 正右方 |
| back-right.png | 右后方 |
| back.png | 正后方 |
| back-left.png | 左后方 |
| left.png | 正左方 |
| front-left.png | 左前方 |
| top-front.png | 前上方 45° |
| top-back.png | 后上方 |
| top-left.png | 左上方 |
| top-right.png | 右上方 |
| top.png | 正上方 |
| bottom.png | 正下方 |

缺少图片时 3D 查看器可能报错；至少需前后左右上下 6 张基础视角可降级展示。
